Advanced Options

Disable User
Disabling a user prevents portal and API access. The user session is immediately destroyed. The user remains assigned to any previously assigned objects. If the user was disabled before, this section allows you to enable the user.
Delete User
Deleting a user removes the user entirely from the database. All audit information is effectively destroyed and cannot be recovered unless you have made a backup.
